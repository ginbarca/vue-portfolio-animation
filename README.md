# VueJS Portfolio Animation

## Project setup
```
yarn install
```
or
```
npm install
```

### Compiles and hot-reloads for development
```
yarn serve
```
or
```
npm run serve
```

### Compiles and minifies for production
```
yarn build
```
or
```
npm run build
```

### Lints and fixes files
```
yarn lint
```
or
```
npm run lint
```

### Deploy to Github Pages
```
yarn deploy
```
or
```
npm run deploy
```
